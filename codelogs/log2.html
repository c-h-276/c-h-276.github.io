Code Log 2: CSS Measuring Units

Explain the difference between absolute and relative CSS length units.

Absolute units specify a value that is fixed at length. This means when the screen resolution changes, the value will stay the same no matter what.

Relative units are units that are not fixed, and will change with a resolution change.

What are pt and px?

"pt" stands for "point", which is 1/72 of an inch.

"px" stands for "pixel", which is 1/96th of an inch.

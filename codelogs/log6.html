    • What did you look up?

    I looked up the tag for list, <"li">

    What are the differences in the information presented on each site?

    Each website contains the same information, but is formatted slightly differently. They all include how the tag aknowleges that something is an item in a list, how this tag would be used to simplify and structure content better, and an example of the code. However, only three included which browsers supported this tag.

    What features do you think a code reference page should offer?

    I think a reference page should include an explination of the code and why it was made, show how the code is used in an example, show similar tags that might be used with this tag, and a series of browsers this tag is supported by.

    Do any of your 4 resources stand out as particularly good or bad?

    None of them stood out as good or bad, as they all gave me the same general basic information about the list tag, however, HTML and W3Schools do make the website hard to read for me as the description of the tag is not up at the top of the page, rather it's near the middle of it.
